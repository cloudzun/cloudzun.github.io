## 👋 你好！

经过一番折腾，我的个人网站终于重新上线了！这次我选择了 [Astro](https://astro.build/) 作为技术栈。

## 为什么选择 Astro？

之前我的网站使用的是 GitHub Pages 的默认 Jekyll 模板，功能比较有限。在考虑了几个方案后，我决定迁移到 Astro：

- **🚀 性能优秀**：Astro 默认生成静态 HTML，加载速度极快
- **📝 Markdown 支持**：原生支持 Markdown 写作，专注于内容创作
- **🎨 灵活定制**：组件化架构，易于扩展和定制
- **🔧 现代化**：支持 TypeScript、Tailwind CSS 等现代工具

## 网站结构

现在的网站包含以下几个主要部分：

- **首页**：个人简介、技术栈、精选项目展示
- **项目页面**：展示我的开源项目和工作
- **博客**：分享技术文章、项目复盘、学习笔记
- **关于页面**：更详细的个人介绍
